About Us

We Code a group of students clubbed together to transfer the legacy of the knowledge and skills they gained during their learning. The club organises classes for students on various subjects related to coding competition, core computer science concepts, programming languages , etc. We Code also organises weekly coding contests for practice.

The club is purely student to student interaction, with no faculty/professor and hesitation, in between students and concepts, where senior university students teaches the concepts to junior students.

  • The founder of We Code is Dr. Ankur Singh Bisht, HOD Computer Science and Engineering, GEHU, Bhimtal Campus.
  • Every year We Code hire new interns to promote the legacy.

We Code is the idea of forming a coding club with some of our seniors, who can teach their juinors about upcomig technologies. They started conducting coding contest and distribute prizes to the top 3 contestants.
They associate We Code with "Universitas Raharja"(Indonasian university) and took the "We Code" to the international level.

Frequently Asked Questions

Commanly asked questions by the students.

  • How many classes should be held in a week?

    We take 6 classes per week with a class of particular domain once a week.

    In we code club, we teaches the technical domain which include Data Structure, Competitive Programming, Web Development, Android Development, Flutter Development, Python and Machine learning, Cyber-Security.

  • Can we learn practicals in our classes?

    Yes, we conduct practical classes separately after theoritical classes.

  • Should we work on projects in we code classes?
